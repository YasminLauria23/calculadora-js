
# 🧮 calculadora-js

Uma calculadora simples e elegante desenvolvida com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Porcentagem
- Inversão de sinal (+/-)
- Ponto flutuante (números decimais)
- Display com expressão e resultado em tempo real
- Botão limpar (C)

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 🎨 Design

Visual escuro e moderno com borda em gradiente azul, seguindo o padrão do portfólio pessoal.

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/calculadora-js.git
```

2. Abra o arquivo `calculadora.html` no navegador.

Pronto! Não precisa instalar nada. 😄
